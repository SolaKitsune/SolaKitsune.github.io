    2026-03-19 第五天#

    今日完成的內容#

    • 把之前日期上的2024改回了2026,我居然一直沒發現，服了
    • 用開開的手機進行測試，發現百度瀏覽器有無法使用書籤高光的BUG（感覺這幾天被高光剋死了），折騰一番後採用了以下方法進行探索：
    1. 嘗試適配（失敗了）
    2. 適配失敗後，嘗試加一個彈框進行提醒（「您使用的瀏覽器可能無法完整支援書籤功能，建議使用 Chrome以獲得最佳體驗」之類的），發現連彈框都跳不出來 3.一怒之下決定選擇古法提示（指之後開一個使用指南頁面然後在裏面直接寫上瀏覽器方面的備註）

    學到的技巧#

    • 適當地放棄

    2026-03-18 第四天#

    今日完成的內容#

    • 修了一下繁簡切換+高光功能上的BUG（由開開發現，具體表現為簡繁切換後會導致文字高光效果消失，而且似乎會導致原本被高光的部分無法導航……儘管被高光的文字還會在書籤列表裏就是）
    • 根據開開的提議，新增了文章開頭的字數統計功能+閱讀進度+剩餘字數功能，並且增加了可以把這玩意收回去變成一個時鐘emoji圖標的功能
    • 把目前更新到的十二章內容全都搬上來了，修了一下第五章的內容，順便因應文裏提到的音樂而插入了相關音樂播放模塊（第四章+五章），並且順便又因此抓出來繁簡切換高光BUG……

    遇到的問題#

    • 繁簡切換高光的那個BUG是一系列的，經過一堆測試後發現，好像但凡是對界面有SCSS改動都會觸發BUG，導致繁簡切換後高光顯示異常。今天大部分時間都花在修這個玩意上了！！！
    • 不小心把那個章節剩餘時間的功能弄沒了，但感覺好像影響也不算很大（畢竟每個人閱讀速度不一樣）

    學到的技巧#

    • 大戰繁簡切換
    • 重啓（在差不多是最後一次關鍵的繁簡切換BUG修復中真的有用）
    • 倔
    • 和BUG抬槓

    待辦事項#

    • 感覺現在網站上的功能圖標有點多了，在想能否弄一個相對比較整潔的統合版
    • 畫廊那邊有點BUG，插圖靠近邊緣的位置會有一部分顯示不出來
    • 看看能不能在文章開頭也加一個切換下一章的鍵？或者類似的功能

    2026-03-17 第三天#

    今日完成的內容#

    • 畫廊！！！！！！！！（雖然目前的感覺效果還不算最好）
    • 對於字體和閱讀界面寬度的三種大小切換（來自開開的提議）
    • 書籤功能（包括選取文字後保存的「熒光筆」與保存整個界面兩個選項），希望有助改善閱讀觀感
    • 書籤列表與管理界面（居然能做到）
    • 章節頁面右下角的上下快速移動摁鍵（應該叫快速導航？但我忍不住稱之為窜天猴）
    • 屏幕最上方的閱讀進度條
    • 閱讀剩餘時間的自動計算
    • 完成 HTML/JS 拆分工程（遇到了一堆BUG）
    • 22:55新增，提示上次閱讀進度的摁鈕
    • 大半夜不睡覺，成功攻克了繁簡切換
    • 大半夜依然不睡覺，嘗試初步弄了雙子的人設介紹界面+性別設定界面

    遇到的問題（現已全部升天）#

    • 書籤點擊時選取範圍消失，用 mousedown 解決了
    • 網站爆炸變成純代碼了（不知道為啥）
    • 快速導航失效（不知道為啥）
    • 左側向導欄的謎之箭頭（後來發現是文件內部開頭寫錯了導致箭頭消失/在不應該有箭頭的地方有箭頭）
    • 人設介紹界面的顏色未能同步更新（用奇怪的方式解決了）

    學到的技巧#

    • 用 mousedown 搶在選取消失前儲存（雖然不知道為啥）
    • 把一堆碼撤銷又註解又開新檔轉移以修復 bug 的方法（雖然也不知道為啥）
    • 備份真的很重要（這個知道了）
    • 運氣
    • 祈禱
    • 重啓
    • 用openCC本地搞繁簡切換

    記錄一個自己感覺很有意思的過程，由於繁體轉簡體是多對一，所以這個功能在用對部署方法後確實比我想像中省事……而關鍵的來了，我要防止它在切換成簡體後再給我機翻成繁體而導致詭異的效果（什麼谷歌翻譯二十次），所以我把它設置成了「原版簡化 → 簡體版」，but 簡體版再換回來時走的步驟就不是繁化了，而是「把字體恢復原狀 → 變回原本的繁體原文」，完全跳過中間那個繁體轉換的階段，於是就沒有 BUG 了

    待辦事項#

    • 繁簡轉換功能（今天上午就在弄了，沒解決，但一定要弄出來）【最新消息：半夜弄出來了，厚厚後厚厚吼吼吼】
    • 章節字數
    • 更新時間
    • 把設定和其他圖和之後的章節搬一下
    • 那個「跳轉到上次的閱讀位置嗎？」的彈框處理得不太好看，之後美化一下（22:55 說時遲那時快，已經解決了，現在新增了一個短暫出現後就會消失的摁鈕用來提示上次的進度）
    • 思考把熱線弄成超文本小説的可能性（是叫這個嗎）
    • 該碼字了（失敗，今天弄完已經01:35了，但姑且是有在做正事所以原諒自己了）
