    阿克韋洛士官長，由他當值的一天

    AM4:23

    他甚至不是因為鬧鐘而醒來，只是身體習慣了這個時間點。茶會結束後，這個星期輪到他當值，離開日晷學院而去其他院校履行學階士官長的其中一種職責。

    學院宿舍的燈仍然關著，走廊裏沒有腳步聲，這是他一天之中唯二不需要對任何人負責的時段，醒來與入睡。

    AM4:30

    起身，整理牀鋪。提供給他的臨時單人宿舍與他熟悉的一切差別不大，唯一不同的那麼一點是，這裏沒有任何屬於他的物品。

    白被子的折疊如何，灰毛毯的整理如何，枕頭的擺放如何……十幾年來都是這個規律，他閉着眼睛也能把一切完成。一個似乎從未有人睡過的牀位很快出現了。

    AM4:36

    幸運地可以用熱水洗漱！開心是一瞬間的事。

    鏡子裏的他看起來比實際年齡更成熟，鏡子裏的那張臉沒有表情。尤莎琳沒有出現。

    AM4:40

    穿戴制服。

    學階士官長的制服細節很多，令他感覺自己把某種身份穿在身上。確保沒有任何可被指摘的地方，他現在已經不會為這種事感到那樣煩惱了。走廊寂靜，大部分人還未醒來。

    AM5:25

    檢查當日名冊與值勤表。夜間違規記錄三起，醫務部臨時請求一項，低年級學員心理評估名單更新……這些都會在早晨點名之前處理完畢。他不覺得那算是「違規」，但他無權質疑，他不是用來提意見的。

    AM5:41

    前往訓練區。

    模擬的晨霧尚未散去，操場空曠。晨練開始前，他會提前巡視一遍器械與場地，確認沒有安全隱患——如果出了事故，責任畢竟會落在他身上。

    AM6:00

    點名時間，他走過學員列隊。沒有人站姿不穩或眼神渙散，這些完美的晝城公民似乎一整排待機時的電子設備那樣整齊。沒有人向他投來視線或聲音，但不悅感如影隨形，會是誰在質疑他、凝視他？往常經驗告訴他，這是永遠找不出來的答案。

    AM6:18

    終於有錯誤在訓練中出現。糾正動作、記錄缺失、口頭警告。他不喜歡處罰別人，但他已經被評為最適合做此類事的那一個。比較令他欣慰的一點在於，

    AM7:00

    訓練結束。學員解散後，他仍留在場地上，檢查器械與遺留物品。

    有一張看似沒有內容的紙條——他確信那是紙條。按照規定而言，不能擅自棄置或者銷毀它，必須上繳。他對其中要傳遞的信息有所預感。

    AM7:10

    簡單的進食，一袋越橘果凍，成分表豐富得令他聯想到字典。這個時間段的他吃得很快，並且很少。從隔了幾個桌位的地方傳來烤土豆的香味。

    AM7:24

    看着人工智能處理行政文件，和人工智能一起工作的感覺不錯——它們會用許多種方式確認同一件事沒有被漏掉。更多的是沒有那種微妙的壓力感，起碼，在行政用人工智能身上沒有。此時他可以負責發呆。

    AM8:03

    陪同一位據稱受到同班同學欺凌的低年級學員前往課室。有人偷偷看他、有人刻意避開視線，他對此從未真正習慣，但他習慣了如何威懾別人。

    找到那張桌子，然後將它，怎麼説，從中間擊裂？再然後説出指定的語句，保證聽者在這之後會於當事人身上聯想到一名兇狠的露斯希亞士官長。好吧，再一次證明了他、以及他所屬的族羣，很擅長此類事情。有時他不得不利用刻板印象。

    AM9:16

    突發狀況，一名學員在實驗課中失控，教官要求他介入。完成得很順利，學員在被擊倒後仍有穩定生命體徵。

    AM10:42

    將學員送往醫務部，填寫報告，補交説明。

    他的名字會出現在文件最後一行，作為「確認人」。之後如果有後續事宜，不論結果好壞與否、走向如何，他都將參與及承擔相關責任——儘管由他負責的事物不會通常不會出什麼差錯，但他早已放棄去期待褒獎了。

    PM12:08

    短暫休息——實際上只是坐下來，把重量從腿上移走。

    其他人可以午休，但他不會睡着。未經許可下的入睡意味著失職。

    PM1:30

    準備下午訓練。

    重新調整場地安排，臨時替換一名請假的助教……好在他對課程內容並不陌生。一切順利。

    PM3:57

    在後操場發生學員衝突，雙方似乎積怨已久。事態並不嚴重，但當然不能放任。不知為何最終變成了他一個人單挑兩人，制服的過程還算順利，兩人在此之後握手言和，仇恨的對象似乎轉移到他身上。

    兩名學員在被擊倒後仍有穩定生命體徵。

    PM5:11

    訓練結束，學院開始變得吵雜……當然，在許可範圍內的程度。對他而言，這意味着另一種工作的開始。

    和其他學階士官長進行半日總結。

    PM6:10

    開始晚間巡查。

    宿舍、走廊、公共區域……確認燈光、門禁、噪音。

    有人向他點頭、主動問好、微笑，有人裝作沒看見，有人眼神微妙。不論如何，他需要回應一切指向他的反應。

    PM7:32

    晚餐，內容依舊簡單。

    他不與學員同桌，也不與教官寒暄……這種距離確實讓工作更加順利。偶爾會見到和他一樣的士官長，不過沒什麼好聊的。

    PM8:15

    處理當日的最終報告。

    學員的違規行為是否需要升級處理？某份文件是否存檔、是否需要提前發送？他和人工智能聊得很開心，有時甚至讓他覺得自己在這些環節上的作用是否僅僅是一個人類陪聊。這樣也不錯，人工智能具有情感需求已經是很久之前就被證明的事了。

    PM9:46

    學院逐漸安靜下來，他需要進行最後一次巡查。確認沒有異常並不是當日工作的尾聲，現在輪到他進行個人的體能訓練。當所有人都認為「露斯希亞人有着發自血脈深處的攻擊性與鬥爭天賦」時，你最好真的有。

    他不需要被誰看見。他是科舒爾倪科夫士官長，他走過學員列隊時沒有人敢直視他，他需要繼續這樣——他需要讓身體繼續成為權威與威懾力的載體，某種強大功能的容器，讓同樣階層的公民之中沒有誰需要、也沒有誰有資格，去注意他裏面裝載了什麼。他厭惡那美麗的茶會，他厭惡被當成某種帶有所謂異域風情的佳釀。

    PM11:02

    回到宿舍。記錄個人備忘……這次不是官方文件，只是他自己的，關於哪些地方需要調整，哪些學員需要更多關注，諸如此類。多些一些被監控所記錄的事。

    PM11:27

    關燈。

    往常而言，黑暗會讓他的呼吸慢下來，但這一次沒有。他有一些事需要告訴自己。如果他不服從——如果他拒絕、反抗、甚至只是表現出明顯的厭惡——他的服從指數會下降，他那有關馴化程度的良好評價會被修改，他的監管等級可能被調整。

    他那僅剩好聽的頭銜，需要維持的公民認可度，永遠低於任何晝城公民的個體權限等級。夠了。他們有要求他配合的資格，而他的配合會被記錄為「服從指數穩定」、「適應性良好」。不要再想了。

    AM00:18

    他終於睡着了。

    不要這樣做，科舒爾倪科夫，不要那樣憤怒。你的身體從來不是「你的」。你是系統的實驗材料，是訓練場的戰鬥人員，是學院的士官長樣本，是權貴茶會上的「swan-ivka」。多一次冒犯，不過是在這長長的清單上再添一筆。

    AM01:19

    抵達訓練場。這是一次重大違規，他的學生生涯中恐怕從未有過這樣對規則的悖逆，但是他執意為自己安排這一刻的槍械訓練，他的Hunting Polka一直在等着這難得的時段。

    槍托的木材被他手掌長年累月的把握打磨出溫潤的光澤，準星在黑暗裡幾乎不可見，但他眼中亮起的熱月足以為他在個人的視野中作充足的照亮，這是一雙為夜晚而生的眼睛。三百公尺步槍三姿，四十發子彈，臥姿、立姿、跪姿。
